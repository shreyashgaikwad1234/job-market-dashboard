# Job Market Dashboard

This project analyzes job market trends using real-world job data.

## Features
- Collected 300+ job listings using API
- Extracted key skills (Python, SQL, Excel, etc.)
- Built data analysis pipeline using Pandas
- Visualized insights using Streamlit dashboard

## Tech Stack
- Python
- Pandas
- Streamlit
- API Integration

## Run Locally
1. Set API token:
   $env:APIFY_TOKEN="your_token"
2. Run:
   python scrapers/apify_fetch.py
